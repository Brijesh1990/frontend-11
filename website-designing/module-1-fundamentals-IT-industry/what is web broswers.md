  **types of website**

  1. static website
     **defintion**
     A static website is website where user can not be change anything else after deploy.
     **examples**
     ```
     http://www.krutionella.com/
     https://www.bhaskar.com/ 
     ```
  2. dynamic website 

      **defintion**
     A dynamic website is website where user can change | add | update | modify   anything else after deploy or on run time i.e called dynamic website.
     **examples**
     ```
     http://www.flipkart.com/
     https://www.amazon.in/
      
     ```
