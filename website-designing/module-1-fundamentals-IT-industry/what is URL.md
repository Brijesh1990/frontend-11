# what is URL ?
  1. URL stands for uniform resource locator
  2. URL is used to locate your website on web broswers
  3. any website content URL
  
  **types of url**

  1. absolute URL
     An absolute url is main url of your website where we can open our Home page or landing.
     **examples**
     ```
      https://www.tops-int.com
       https://www.flipkart.com
     ``` 
  2. relative URL
      An absolute url is main url of your website where we can open our Home page or landing.
      
      **examples**
     ```
     https://www.tops-int.com/about-us
     https://www.tops-int.com/contact-us
     https://www.flipkart/about-us   
     ```
