# what is Protocol ?
  1. protocol is a set of rules i.e called protocol
  2. protocol is used to set rules in your webiste how to access website

  
  **types of protocol**

  1. http
     access with http it not secured 
     http stands for hypertext transfer protocol

    **examples**
     ```
      http://www.tops-int.com
      http://www.krutionnela.com
     ``` 
  2. https
     access with http it fully secured 
     https stands for secured hypertext transfer protocol
 
      **examples**
     ```
     https://www.tops-int.com/
     https://www.tops-int.com/contact-us
     https://www.flipkart/about-us   
     ```
