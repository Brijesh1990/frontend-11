# what is github ?
  1. github is a product of microsoft
  2. github is a server where we stored our data and shared with anyone
  3. github is an repository systems
  4. github is used to shared data | commit data | add data | push data | pull data on its server 
  5. how to setup github ....

  **How to setup github**
  
   1. create an account on github
      https://github.com/

   2. create an repository 
      https://github.com/Brijesh1990/frontend-11.git

   3. download git software or gitbash we used this to apply commands to upload data on github server.

      https://git-scm.com/install/windows

   4. open git bash form start menu     

   5. write a commands on gitbash 

    1. git init
    2. git add .
    3. git commit -m "first commit"
    4. git remote add origin https://github.com/Brijesh1990/frontend-11.git
    5. git push -u origin master
