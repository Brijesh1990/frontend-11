## html tags 

1. html provides some types of tags 
2. html tags create an structures
     **examples**
     ```
     <p>Hey brijesh</p>
     <h1>Hey brijesh</h1>

     ````
3. types of tags
    1. paired tags
       Note : paired tags are start and must be closed i.e called paired 

        **examples**

        ```
        <p></p>
        <h1></h1>
        <form></form>
        <table></table>
        <frameset></frameset>
        <frame></frame>
        <iframe</iframe>

        ```
    2. non-paired tags    

        Note : non-paired tags are start but not need to closed  i.e called non-paired tags 

        **examples**
        ```
        <br>
        <hr>
        <link>
        <meta>
        <img>

        ```
